# WEB-LLM with @mlc-ai/web-llm
## Copy of Google Repo (Todo AI)

This is a demo application for a to-do list, built for the on-device AI blog post series.

Reference: [Link to Article](https://web.dev/articles/ai-chatbot-webllm)

It was copied from the second branch of the original repository. The original branches are:

1. `main`: normal to-do app
2. `web-llm`: AI chatbot using WebLLM
3. `prompt-api`: AI chatbot using Prompt API

To run this demo, please clone this repository and run the following commands:

```shell
npm i
npm start
```
